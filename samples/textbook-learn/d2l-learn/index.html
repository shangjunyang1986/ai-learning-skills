3 · 线性回归 逐章精讲开始

最简单、也最该先懂的模型：用特征的加权和 + 偏置预测一个连续值。它是理解后面一切的「单神经元」原型。

核心要点（忠于原书）

  • 模型是仿射函数：预测 = 各特征的加权和再加偏置。
  • 用平方损失衡量好坏，训练就是在数据上最小化平均损失。
  • 平方损失下存在罕见的解析解（正规方程），但一般我们用小批量随机梯度下降 (SGD)。
  • 平方损失不是随便选的——它等价于「加性高斯噪声」假设下的极大似然估计。

关键公式

ŷ = w⊤x + b (3.1.3) 单样本预测
L(w,b) = (1/n) Σi ½ (w⊤x(i) + b − y(i))2 (3.1.6) 平均平方损失
w* = (X⊤X)−1 X⊤y (3.1.9) 正规方程 / 解析解
出处：3.1 Linear Regression（公式编号同书）。
📝例题精讲 · 推导正规方程

把损失写成 ‖y − Xw‖²（把偏置 b 并入 w，对应 X 补一列 1）。求使损失最小的 w。

▸ 展开解答
  1. 写出目标：最小化 ‖y − Xw‖² = (y−Xw)⊤(y−Xw)。
  2. 对 w 求梯度：∂w‖y − Xw‖² = 2X⊤(Xw − y)。
  3. 令梯度为零：X⊤(Xw − y) = 0 ⟹ X⊤Xw = X⊤y。
  4. 解出：w* = (X⊤X)−1X⊤y。
关键前提：只有当 X⊤X 可逆（设计矩阵满秩）时成立。若不满秩，需正则化/加噪等手段——这正是原书练习 4 让你思考的。
🧠主动回忆 · 线性回归

Q1为什么说平方损失「有统计学道理」，而不是随便选的？

B。原书 3.1 证明：在 y = w⊤x + b + ε、ε~高斯 的假设下，最小化平方损失 = 最大化似然。鲁棒于异常值的反而是绝对值损失（练习 1c）。

Q2正规方程给出的解析解是？

A。由 X⊤Xw = X⊤y 解出，前提是 X⊤X 满秩可逆。
线性回归什么时候有解析解？
当 X⊤X 满秩（可逆）时；否则只能靠正则化或迭代优化。

一般情况下我们用什么方法训练它？
小批量随机梯度下降（minibatch SGD），更新式 3.1.10。

4 · Softmax 分类

从「预测一个数」跨到「预测哪一类」：用 softmax 把原始分数变成概率分布，用交叉熵当损失。

核心要点

  • 分类问题问的是「哪一个」，标签用 one-hot 编码，每个类别一个输出。
  • softmax 把 logits 变成非负、和为 1 的概率，同时保持 argmax 不变。
  • 交叉熵 = 正确类别的负对数似然，是分类的自然损失。
  • softmax + 交叉熵的梯度形式极简：预测概率 − 真实标签。

关键公式

ŷi = exp(oi) / Σj exp(oj) (4.1.3) softmax
l(y, ŷ) = − Σj yj log ŷj (4.1.8) 交叉熵
∂ l / ∂ oj = softmax(o)j − yj (4.1.10) 干净的梯度
出处：4.1 Softmax Regression。
📝例题精讲 · 手算一遍 softmax + 交叉熵

设 logits o = (2, 1, 0)，真实类别是第 0 类（one-hot (1,0,0)）。求 softmax 概率、交叉熵损失、以及对 o 的梯度。

▸ 展开解答
  1. 取指数：exp(2),exp(1),exp(0) ≈ (7.389, 2.718, 1.000)，和 ≈ 11.107。
  2. softmax：(7.389, 2.718, 1.000)/11.107 ≈ (0.665, 0.245, 0.090)。
  3. 交叉熵：只有正确类有贡献，l = −log(0.665) ≈ 0.408。
  4. 梯度 = softmax − y：(0.665−1, 0.245−0, 0.090−0) = (−0.335, 0.245, 0.090)。
读法：梯度告诉你——把正确类的 logit 往上推（负梯度方向），把另两类往下压，幅度正比于它们「冒头」的概率。（这组具体数字是按书中公式 4.1.3/4.1.8/4.1.10 的演算示例，书里并未印出这几个数。）
🧠主动回忆 · Softmax 分类

Q1softmax+交叉熵 对某个 logit oj 的梯度等于？

C。正是公式 4.1.10——这种「预测 − 真实」的干净形式，是 softmax 配交叉熵被广泛采用的原因之一。

Q2为什么要套一层 softmax，而不直接拿 logits 当输出？

B。softmax 保持 argmax 不变，只是把任意实数分数规范成合法概率分布，方便用交叉熵和做概率解释。它本身不改变模型是「线性」这一事实。
交叉熵损失 l(y, ŷ) 的式子？
l = − Σⱼ yⱼ log ŷⱼ —— 因 y 是 one-hot，等于正确类预测概率的负对数。
点击翻面看答案
分类的标签怎么编码？输出层有几个单元？
one-hot 编码；每个类别一个输出单元（q 类就 q 个输出），全连接到输入。

5 · 多层感知机 (MLP)

给网络加上隐藏层和非线性激活——这是从「线性模型」迈向「深度」的第一步，也是关键一步。

核心要点

  • 线性模型假设太强（单调/线性），现实问题常常不成立。
  • 隐藏层在输入与输出间加入中间表示。
  • 没有非线性，多个仿射层会「坍缩」成单个仿射层——白加。
  • ReLU / sigmoid / tanh 等激活才让「深度」真正有意义；单隐藏层 MLP 即万能逼近器。

关键公式

无激活：O = (XW(1)+b(1))W(2)+b(2) = XW + b (5.1.2) 坍缩
加激活：H = σ(XW(1)+b(1)), O = HW(2)+b(2) (5.1.3) 真·MLP
ReLU(x) = max(x, 0) sigmoid(x) = 1/(1+e−x) (5.1.4 / 5.1.6)
出处：5.1 Multilayer Perceptrons。
📝例题精讲 · 为什么「纯线性堆叠」等于白堆

两层都不加激活：H = XW(1)+b(1)，O = HW(2)+b(2)。证明它和单个线性层表达力相同。

▸ 展开解答
  1. 代入：O = (XW(1)+b(1))W(2) + b(2)。
  2. 展开：O = X(W(1)W(2)) + (b(1)W(2)+b(2))。
  3. 令 W = W(1)W(2)、b = b(1)W(2)+b(2)，则 O = XW + b。
  4. 结论：两层线性网络 = 单个线性层，「白忙一场」。插入非线性 σ（式 5.1.3）才打破坍缩。
一句话记住：仿射的仿射还是仿射。深度要靠非线性才换来表达力。（原书练习 1）
🧠主动回忆 · MLP

Q1给一个「纯线性」深网（层与层之间无激活）不断加层，会怎样？

B。仿射复合仍是仿射（例题已证）。原书练习 1 还要你举出「加层反而降低表达力」的例子。

Q2下面哪个恒等式成立？（原书练习 5a）

A。这说明 sigmoid 与 tanh 只差一个仿射的前后处理，二者参数化的函数类其实相同（练习 5b）。
ReLU(x) 的定义？它解决了什么？
ReLU(x)=max(x,0)。计算简单、正区间梯度恒为 1，缓解了 sigmoid 的梯度消失。

7 · 卷积神经网络 书第 7 章

专门「看图」的架构。它不是凭空发明的——把平移不变性和局部性两条原则加到全连接层上，自然就推出了卷积。

核心要点

  • 全连接层撑不起图像：百万像素接千单元隐藏层 ≈ 109 个参数。
  • 平移不变性：图像平移，表示也应随之平移——权重不该依赖绝对位置 ⟹ 权重共享。
  • 局部性：一个隐藏单元只看输入的一小块邻域 ⟹ 有界感受野。
  • 两条原则一加，参数量直降十个数量级——结果正好就是卷积层；真实图像还要加通道。

关键公式

[H]i,j = u + Σa=−Δ..Δ Σb=−Δ..Δ [V]a,b [X]i+a, j+b (7.1.3) 加上两原则后
输出尺寸 = (nh − kh + 1) × (nw − kw + 1) (7.2) 无填充无步幅时
出处：7.1 从全连接到卷积 与 7.2 图像卷积。
📝例题精讲 · 手算一次二维互相关

输入 [[0,1,2],[3,4,5],[6,7,8]]，核 [[0,1],[2,3]]。求输出（步幅 1、无填充）。

▸ 展开解答
  1. 输出尺寸：(3−2+1)×(3−2+1) = 2×2。
  2. 左上：0·0+1·1+3·2+4·3 = 19。
  3. 右上：1·0+2·1+4·2+5·3 = 25。
  4. 左下：3·0+4·1+6·2+7·3 = 37。
  5. 右下：4·0+5·1+7·2+8·3 = 43，得 [[19,25],[37,43]]。
核在输入上滑动，每个位置做「逐元素相乘再求和」。这就是图 7.2.1 在算的事（原书式 7.2.1）。
🧠主动回忆 · 卷积

Q1卷积层是从全连接层施加哪两条原则「推导」出来的？

C。平移不变性 ⟹ 权重共享（V 不依赖位置）；局部性 ⟹ 只看 ±Δ 邻域。两者一起，全连接层就退化成卷积层。

Q2上面例题里输出的左上角元素是多少？

A。0·0+1·1+3·2+4·3 = 19。
无填充、步幅 1 时，卷积输出尺寸公式？
(n_h − k_h + 1) × (n_w − k_w + 1)。
点击翻面看答案
深度学习的「卷积」和数学上的卷积差在哪？
实现的其实是互相关（核不翻转，用 +a,+b）。因核是学出来的，翻不翻转不影响结果，所以习惯上仍叫卷积。
